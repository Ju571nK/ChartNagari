## 1. Options Flow API

옵션 플로우 데이터는 비정상적으로 큰 옵션 거래(스윕, 블록 등)를 실시간으로 감지하는 데이터입니다.

### 1-1. Unusual Whales

| 플랜 | 가격 | 비고 |
|------|------|------|
| Free | $0/월 | API 없음, 제한된 플랫폼 접근 |
| Base (플랫폼) | $44/월 (연간), $50/월 (월간) | 다크풀 피드 포함 |
| API (별도) | ~$60+/월 (2025년 5월 인상) | 정확한 금액은 로그인 후 확인 |
| Enterprise | 별도 문의 | 대량 피드, 커스텀 SLA |

**포함 데이터:**
- 실시간 옵션 플로우 (OPRA 소스)
- 다크풀 프린트 (15분 지연, 8,000+ 티커)
- 의회 거래 데이터 (독보적 차별점)
- 기관 보유 현황 (13F)
- 100+ REST 엔드포인트

**주요 특징:**
- MCP 서버 지원 (AI 에이전트 통합)
- 0DTE 플로우 추적
- 어닝스, IPO, 매크로 캘린더 포함
- Discord 봇 추가($10/월)

**장점:** 단일 구독으로 플로우+다크풀+대안 데이터 제공, 가장 낮은 진입 장벽
**단점:** API 정확한 가격 불투명(로그인 필요), 다크풀 15분 지연

## 2. Stock Data API

| Business | $1,999/월 | 실시간 + 재배포권 | 20년+ |

**포함 데이터:**
- 미국 주식, ETF, 옵션, 외환, 암호화폐, 선물(CME/CBOT/NYMEX)
- WebSocket: 거래, 호가, 분/초 단위 집계

**단점:** 미국 전용(글로벌 거래소 없음), 재무제표 없음(SEC 원시 공시만), 실시간은 $199/월 필요

## 4. Dark Pool API

다크풀 데이터는 장외 거래소(ATS: Alternative Trading System)에서 체결된 기관 블록 거래를 추적합니다.
⚠️ **중요:** 규정상 소매 제공자의 다크풀 프린트는 항상 **~15분 지연**됩니다.

### 4-1. Unusual Whales (다크풀 포함)

| 플랜 | 가격 |
|------|------|
| Base 플랫폼 | $44/월 (연간) |
| API (별도) | $60+/월 (2025년 5월 인상 후) |
| Enterprise | 별도 문의 |

**포함 데이터:**
- 8,000+ 티커 다크풀 프린트 (~15분 지연)
- 다크풀 레벨 (기관 지지/저항선 집계)
- REST API 100+ 엔드포인트
- 히스토리 다크풀 피드 (Data Shop에서 별도 구매)
- MCP 서버 지원

**장점:** 옵션 플로우 + 다크풀 + 대안 데이터 통합, API 접근 가능
**단점:** 프린트 15분 지연, API 가격 불투명

### 4-4. FINRA OTC Transparency API (무료 공식 데이터)

| 항목 | 내용 |
|------|------|
| 가격 | **무료** |
| 데이터 지연 | Tier 1 NMS (S&P 500, Russell 1000): **2주** / 기타: **4주** |
| 히스토리 | 4년 (weeklySummaryHistoric) |

**API 정보:**
- 엔드포인트: `https://api.finra.org/data/group/otcMarket/name/weeklySummary`
- 응답 형식: JSON / CSV
- 페이지네이션: `limit` (최대 5,000) + `offset`
- 인증: 무료 등록 필요 (developer.finra.org)
- OpenBB 통합: `obb.equity.darkpool.otc(provider='finra')`

**포함 데이터:**
- ATS별, 종목별 주간 거래량 + 거래 횟수
- 블록 거래 집계 데이터 (월간)
- 비ATS OTC 거래량
- 어떤 ATS(IEX, Liquidnet 등)가 어떤 주식을 거래했는지 파악 가능

**장점:** 무료, 공식 규제 데이터, 거래소별 세분화, 4년 히스토리
**단점:** 2~4주 지연으로 실시간 트레이딩 불가, 개별 프린트 없음(주간 집계만), 암호화폐 없음
